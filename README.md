# Taros Chat Widget for WordPress

Official WordPress plugin for [Taros](https://taros.ai) - Add an AI-powered chat widget to your WordPress or WooCommerce store.

## Installation

### From WordPress.org (Recommended)
1. Go to Plugins → Add New in your WordPress dashboard
2. Search for "Taros Chat Widget"
3. Click Install → Activate
4. Go to Settings → Taros Chat and enter your Bot ID

### Manual Installation
1. Download the latest release
2. Upload to `/wp-content/plugins/taros-chat-widget/`
3. Activate the plugin
4. Go to Settings → Taros Chat and enter your Bot ID

## Getting Your Bot ID

1. Create a free account at [taros.ai](https://taros.ai)
2. Create a chatbot and train it with your content
3. Go to Widget settings in your dashboard
4. Copy your Bot ID

## Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
